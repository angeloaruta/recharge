# @recharge/tsconfig

Shared TypeScript configurations for the Recharge monorepo.

## Usage

### Base Config

```json
{
  "extends": "@recharge/tsconfig/base.json"
}
```

### Next.js Config

```json
{
  "extends": "@recharge/tsconfig/nextjs.json"
}
```

### React Library Config

```json
{
  "extends": "@recharge/tsconfig/react-library.json"
}
```
